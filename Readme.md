# Corg-Labs Organization Configuration

This repository contains **organization-wide configuration files** for Corg-Labs.

The files here apply by default to all repositories in the organization unless explicitly overridden.

---

## Contents

- `profile/README.md`  
  → Public-facing organization landing page

- `CONTRIBUTING.md`  
  → Contribution guidelines shared across all repositories

- `CODE_OF_CONDUCT.md`  
  → Behavioral expectations for contributors

- `SECURITY.md`
  → Security policy for reporting Issues and vulnerabilities
